HTML 元素

HTML 元素（Element）是由 HTML 标签定义的

HTML 元素

HTML 元素指的是开始标签（start tag）到结束标签（end tag）中的所有代码。

开始标签 元素内容 结束标签
<p> This is a paragraph </p>
<a href="https://teemofanfan.github.io"> This is a link </a>
<br />

注释：开始标签常被称为开放标签（opening tag），结束标签常被称为闭合标签（closing tag）。

HTML 元素语法

  • HTML 元素以开始标签起始
  • HTML 元素以结束标签终止
  • 元素的内容是开始标签与结束标签之间的内容
  • 某些 HTML 元素具有空内容（empty content）
  • 空元素在开始标签中进行关闭 [即以开始标签的结束而结束(开始就是结束)]
  • 大多数 HTML 元素可拥有属性

提示：你将在本教程的下一章中学习更多有关属性的内容。

<p> 元素：

这个<p>元素定义了 HTML 文档中的一个段落。

这个<body>元素定义了 HTML 文档的主体（即内容）。

<html> 元素：

这个<html>元素定义了整个 HTML 文档。
这个元素拥有一个开始标签<html>以及一个结束标签</html>。
元素内容为另两个 HTML 元素，<body>元素和<p>元素。
<body>元素拥有一个开始标签<body>以及一个结束标签</body>，
元素内容为：<p>This is a paragraph.</p>。
<p>元素拥有一个开始标签<p>以及一个结束标签</p>，
元素内容为：This is a paragraph.

不要忘记结束标签（随手关门）

即使你没有关门，浏览器依旧会正常并正确地显示 HTML ：

如上，即使你只有开始标签，没有结束标签

空的 HTML 元素

没有内容的 HTML 元素被称为空元素。空元素是在开始标签中关闭的(开始就是结束)。
<br>就是没有关闭标签的空元素（<br>标签定义换行）。
在 XHTML、XML 以及未来版本的 HTML 中，所有元素都必须关闭。
在<br>开始标签中添加斜杠/（结束标志），如<br />，是关闭空元素的正确方法，
HTML、XHTML 和 XML 都接受这种方式。
即使<br>在所有浏览器中都是有效的，但是使用<br />具有更长远的保障。
（否则如果更新了的话，你又要把所有的<br>替换成<br />，不管你想不想，我反正不想。）

HTML 提示：使用小写标签

HTML 标签对大小写不敏感，如<P>和<p>在浏览器中都能正确显示。
但推荐使用小写标签，因为在未来 (X)HTML 版本中强制使用小写标签。
